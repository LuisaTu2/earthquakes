## Get started
1. Clone the project 
2. Navigate to the following files and add your own Google Maps API key in the map script: <br>
    public>>templates>>map.html <br>
    views>>home.hbs <br>

![Insert gKey](gKey.PNG)

3. Run npm install
4. From the root folder run:
node server.js
5. Run program on local host 8080
